# Call List

Тестовое задание: таблица звонков с фильтрацией и воспроизведением записей.

## Стек

- **React 19** + **TypeScript**
- **Redux Toolkit** + **RTK Query** — управление состоянием и запросы к API
- **SCSS Modules** — стилизация
- **Vite** — сборщик
- **Feature-Sliced Design (FSD)** — архитектура проекта

## Функциональность

- Список звонков
- Фильтрация по типу звонка (входящие / исходящие)
- Получение и воспроизведение файла записи звонка

## Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173)

## Сборка

```bash
npm run build
npm run preview
```
